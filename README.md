⚔️ DinoQuest
"Turn life into quests. Turn progress into power."

DinoQuest is a gamified RPG platform designed to transform daily habits, learning, and challenges into an immersive progression system. Originally forked from ChoreQuest, this evolution strips away the "chore" terminology in favor of a unified quest engine, pet-leveling mechanics, and a bold Neo-Brutalism UI.

🚀 The Core Loop
Quests ➔ XP ➔ Levels ➔ Pets/Rewards ➔ New Quests

✨ Features
🗡️ Unified Quest Engine
Dynamic Categories: Daily Quests, Learning Missions, Creative Sprints, and Epic Challenges.

Quest Calendar: Visual scheduling to manage streaks and upcoming adventures.

🧠 Integrated Learning System
Tests & Quizzes: Timed challenges and multiple-choice tests integrated directly into quests.

Adaptive Scoring: XP is awarded based on accuracy, speed, and personal improvement.

Skill Insights: Track performance across different knowledge domains.

🐾 RPG Progression
Pet System: Hatch, level up, and evolve digital companions using earned XP.

Avatar Customization: Unlock gear and styles through rank progression.

XP Reward Shop: A parent-managed storefront for real-world or digital rewards.

🎉 Family Hub
Leaderboards: Friendly competition with siblings or peers.

Social Layer: Shoutouts, emotes, and achievement broadcasting (Bronze, Silver, Gold).

🧱 Design Language: Neo-Brutalism
The UI is built to be "playfully aggressive" and highly legible for kids:

High Contrast: Stark black borders (border-4) and vibrant "pop" colors.

Hard Shadows: Deep, non-blurred box shadows for a tactile 3D feel.

Structured Layouts: Minimal gradients, sharp corners, and a strong visual hierarchy.

UX: Mobile-first design, retained as a PWA for "app-like" home screen access.
